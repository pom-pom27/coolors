## Features
- [x] Authentication with Clerk.
- [x] Generate palettes.
- [x] Save selected palettes and color to user database.
- [x] Exporting palettes as PNG, PDF and SVG.


## Stack
- Nextjs
- Tailwind
- shadcn/UI
- React Hook Form & Zod
- Framer Motion
- Clerk
- Supabase
